# My paper notes


## 1. 阅读流程 (Three‑Pass + Anchored Skim)

| 阶段 | 目标 | 行动 | 产出 |
|------|------|------|------|
| **0. Anchor Skim (< 5 min)** | 判断是否值得精读 | 读标题、作者单位、Abstract、最后一段 Conclusion；浏览图表与算法框图 | “Go / Pass” 标签 |
| **1st Pass (≈ 10 min)** | 建立大局观 | 粗读 Introduction & Conclusion；扫描 Section 标题 | Note 中填 **TL;DR、问题定义、主要贡献** |
| **2nd Pass (≈ 30 min)** | 抓核心方法 | 详读 Method & Key Figures；梳理方程推导思路 | Note 中填 **方法流程、伪代码** |
| **3rd Pass (≥ 30 min)** | 评估可复现性与应用价值 | 深读实验设置、消融、局限；比对相关工作 | Note 中填 **实验配置、结果表格、局限 & 思考** |
| **Post‑Read (当日内)** | 巩固 / 链接 | 将笔记链接到知识网络；为关键要点写卡片 | 间隔复习条目 & 双向链接 |

> **要点：** 严格计时，宁可下次再深读也不要一次性“沉没”在细节里。
